## Updating Documentation

To add or update images, submit a PR to the `main` branch with a YAML config file:

1. Create a YAML file in [docs/src/data/](https://github.com/aws/deep-learning-containers/tree/main/docs/src/data)`<your-repository>/`
2. Include fields that will display in your image tables (see [docs/src/tables/](https://github.com/aws/deep-learning-containers/tree/main/docs/src/tables) for table configurations)
3. Submit a PR to the `main` branch
