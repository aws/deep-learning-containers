# AWS Deep Learning Containers for TensorFlow 2.18 Inference on EC2, ECS, and EKS

[AWS Deep Learning Containers](https://aws.amazon.com/machine-learning/containers/) (DLC) for Amazon EC2, ECS, and EKS are now available with support for TensorFlow 2.18 with CUDA 12.2 on Ubuntu 20.04.

## Release Notes

- Introduced containers for TensorFlow 2.18 for EC2, ECS, EKS
- For more details on TensorFlow 2.18 EC2, ECS, EKS Inference DLC, please refer to [v1.0-tf-ec2-2.18.0-inf-py310](https://github.com/aws/deep-learning-containers/releases/tag/v1.0-tf-ec2-2.18.0-inf-py310).

## Security Advisory

AWS recommends that customers monitor critical security updates in the [AWS Security Bulletin](https://aws.amazon.com/security/security-bulletins/).

## Python Support

Python 3.10 is supported in the containers.

## GPU Instance Type Support

- CUDA 12.2
- cuDNN 8.9.4.25-1+cuda12.2
- NCCL 2.21.5-1+cuda12.2
