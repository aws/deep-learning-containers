# AWS Deep Learning Containers for vLLM 0.14.0 on SageMaker

AWS Deep Learning Containers for SageMaker are now available with vLLM 0.14.0.

## Announcements

- Introduced vLLM 0.14.0 containers for SageMaker

- Added Python 3.12 support

## Core Packages

| Package | Version |
| --- | --- |
| vLLM | 0.14.0 |
| PyTorch | 2.9.1 |
| TorchVision | 0.24.1 |
| TorchAudio | 2.9.1 |
| Transformers | 4.57.6 |
| CUDA | 12.9.1 |
| cuDNN | 9.15.0 |
| NCCL | 2.28.3 |
| EFA | 1.46.0 |

## Security Advisory

AWS recommends that customers monitor critical security updates in the [AWS Security Bulletin](https://aws.amazon.com/security/security-bulletins/).
