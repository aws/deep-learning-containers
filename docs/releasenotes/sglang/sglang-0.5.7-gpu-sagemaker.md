# AWS Deep Learning Containers for SGLang 0.5.7 on SageMaker

AWS Deep Learning Containers for SageMaker are now available with SGLang 0.5.7.

## Announcements

- Introduced SGLang 0.5.7 containers for SageMaker

- Added Python 3.12 support

- Ubuntu 24.04 support

## Core Packages

| Package | Version |
| --- | --- |
| SGLang | 0.5.7 |
| PyTorch | 2.9.1 |
| TorchVision | 0.24.1 |
| TorchAudio | 2.9.1 |
| Transformers | 4.57.1 |
| CUDA | 12.9.1 |
| cuDNN | 9.16.0.29 |
| NCCL | 2.28.3 |
| EFA | 1.46.0 |

## Security Advisory

AWS recommends that customers monitor critical security updates in the [AWS Security Bulletin](https://aws.amazon.com/security/security-bulletins/).
