# AWS Deep Learning Containers for PyTorch 2.8 Training on EC2, ECS and EKS

[AWS Deep Learning Containers](https://aws.amazon.com/machine-learning/containers/) (DLCs) for Amazon EC2, ECS and EKS are now available with PyTorch
2.8 and support for CUDA 12.9 on Ubuntu 22.04.

## Release Notes

- Introduced containers for PyTorch 2.8 for Training which support EC2, ECS and EKS. For details about this release, check out our GitHub
  [release tag](https://github.com/aws/deep-learning-containers/releases/tag/v1.0-pt-ec2-2.8.0-tr-py312).
- Added Python 3.12 support
- Added PyTorch domain libraries: torchtnt 0.2.4, torchdata 0.11.0, torchaudio 2.8.0, torchvision 0.23.0
- Added CUDA 12.9, Ubuntu 22.04 support

## Security Advisory

AWS recommends that customers monitor critical security updates in the [AWS Security Bulletin](https://aws.amazon.com/security/security-bulletins/).

## GPU Instance Type Support

- CUDA 12.9
- cuDNN 9.10.2.21
- NCCL 2.27.3-1
