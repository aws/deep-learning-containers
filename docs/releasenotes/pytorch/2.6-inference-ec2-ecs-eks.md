# AWS Deep Learning Containers for PyTorch 2.6 Inference on EC2, ECS and EKS

[AWS Deep Learning Containers](https://aws.amazon.com/machine-learning/containers/) (DLCs) for Amazon EC2, ECS, and EKS are now available with PyTorch
2.6 and support for CUDA 12.4 on Ubuntu 22.04.

## Release Notes

- Introduced containers for PyTorch 2.6.0 for inference supporting EC2, ECS, and EKS. For details about this release, check out our GitHub
  [release tag](https://github.com/aws/deep-learning-containers/releases/tag/v1.0-pt-ec2-2.6.0-inf-py312).
- Starting with PyTorch 2.6, we are removing Conda from the DLCs and installing all Python packages from PyPI.

## Security Advisory

AWS recommends that customers monitor critical security updates in the [AWS Security Bulletin](https://aws.amazon.com/security/security-bulletins/).

## GPU Instance Type Support

- CUDA 12.4.1
- cuDNN 9.1.0.70+cuda12.4
- NCCL 2.23.4+cuda12.4
